▪ чем отличается pull от fetch?
команда fetch подтягивает все объекты из ремоута,
но не пытается их слить с локальными изменениями.
Это отличает ее от команды git pull, при
использовании которой слияние происходит
автоматически.
используйте fetch если хотите притянуть объекты но
опасаетесь что слияние кода будет чревато
конфликтами.
▪ как создать локальную ветку и переключиться на неё?
git checkout mybranch # переключится на ветку mybranch
git checkout -b mybranch # создаст ветку mybranch и переключится на неё
git switch mybranch # переключится на ветку mybranch НАЧИНАЯ С ВЕРСИИ git 2.23
▪ что будет с веткой в ремоуте если удалить локальную ветку?
Удаление ветки локально не означает удаление
ветки в ремоуте, пока вы это не запушите.
В ремоуте ничего не поменяется пока вы не
запушите какие-либо изменения. В том числе
удаление ветки из ремоута
