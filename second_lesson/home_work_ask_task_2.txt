▪ В каких случаях используется «for»?
Чтобы поочередно подставлять элементы какой-либо последовательности в тело цикла, используется for.
Переменная, которая используется в цикле, будет поочерёдно принимать значения из этой последовательности.
Такая переменная называется «переменной цикла».

▪ В каких случаях используется «while»?
Для цикла while нужно условие.
Тело этого цикла выполняется, пока верно условие.
Условием называется выражение, которое можно интерпретировать как True или False.
Когда условие «True» - тело цикла выполняется.
Цикл while имеет смысл использовать, когда количество его итераций заранее не известно.

▪ Что такое итератор?
Итератор — объект, который содержит исчисляемое количество элементов и по которому можно пройтись поочерёдно.
Любой объект в python, для которого реализован метод __next__() - итератор.
Итерация — это повторяющееся исполнение набора выражений.

▪ Чем итерируемый объект отличается от итератора?
Генератор отличается от итерируемого объекта тем, что не хранит элементы в памяти.
Он генерирует их при вызове.

▪ В чем преимущество использования генераторов?
Эта особенность генераторов вынуждает ими пользоваться при итерации через большие наборы результатов.
